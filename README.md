

Pune Real Estate Market Analysis

Project Overview

This project analyzes the Pune real estate market using property data to understand pricing trends and market insights. The goal of this project is to demonstrate the complete data analysis workflow including data cleaning, querying, and visualization.

Project Workflow

1. Data Cleaning (Python)

- Removed duplicate records
- Handled missing values
- Standardized column formats
- Prepared the dataset for analysis using Pandas

2. Data Analysis (SQL)

SQL queries were used to calculate important metrics:

- Average property price
- Maximum and minimum property prices
- Total number of properties
- Price analysis based on bedrooms, furnishing type, and possession status

3. Data Visualization

An interactive dashboard was created to visualize:

- Average price per square foot
- Property price distribution
- Price vs built-up area
- Average price by bedrooms
- Property distribution by possession status

Key Insights

- Most properties in Pune are Ready to Move
- Property price increases with built-up area and number of bedrooms
- Fully furnished properties have higher average prices

Tools Used

- Python (Pandas, NumPy)
- SQL
- Data Visualization Dashboard

Project Outcome

This project demonstrates how raw real estate data can be transformed into meaningful insights using data analytics techniques.
